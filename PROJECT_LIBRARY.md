## Main portfolio picks

### SpotOnSight

**One-line version**

SpotOnSight is a location-based social platform for discovering, saving, and sharing places across web and mobile clients.

**Portfolio-ready description**

Built a full-stack location-based social platform with a Vue frontend, FastAPI backend, MongoDB persistence, mobile wrapping through Capacitor, Docker-based environments, and CI/CD automation. The project shows I can structure a larger codebase, separate frontend and backend responsibilities, document architecture decisions, and support development with tests and deployment workflows.

**What this proves**

- Full-stack application design
- API and backend structuring
- Frontend architecture and routing
- Environment and deployment setup
- Product thinking beyond a single feature

**Best angles to mention**

- Map or social discovery app
- Multi-environment development workflow
- Test-backed and documented engineering work
- Product-style repo organization

### pay-qr-with-apple-pay

**One-line version**

pay-qr-with-apple-pay is a prototype app that scans QR invoice data and routes the payment flow through Apple Pay and Stripe.

**Portfolio-ready description**

Built a full-stack prototype for QR-based invoice payment with a Vue 3 frontend and Express backend, including invoice parsing, Stripe-powered Apple Pay flow, demo-mode support for unsupported devices, and architecture documentation. This project shows practical product thinking, payment-flow design, and the ability to build a user-focused prototype around a real-world use case.

**What this proves**

- Frontend to backend workflow design
- Payment integration concepts
- User-flow and prototype thinking
- Modular documentation and environment setup

**Best angles to mention**

- Fintech-style prototype
- Real-world payment journey
- Practical demo-ready app design

### better_weather

better_weather looks like a weather/data platform with a strong focus on runtime architecture and testability.

Developed a multi-part weather and data-handling project that combines Python services with a frontend and a registry-based architecture designed for testability.
